# Location-Based Social Media Usage Analysis

## Motivation
I aim to analyze how my social media consumption patterns (YouTube, Instagram, and TikTok) vary based on my location. This analysis will help me understand my digital habits in different environments and potentially optimize my social media usage.

## Dataset Description
I will collect and analyze the following data:

1. Location Data (Google Timeline):
- Geographic locations I visit
- Time spent at each location
- Types of places (home, university, cafes, etc.)

2. Social Media Usage Data:
- YouTube watch history and duration
- Instagram usage patterns
- TikTok viewing history
- All with timestamps for location correlation

## Project Plan

### Data Collection
- Export location history from Google Timeline
- Download activity data from social media platforms
- Focus on the last 3 months of data

### Analysis Methods
- Merge location and usage data using timestamps
- Create visualizations of usage patterns by location
- Identify peak usage locations and times
- Analyze platform preferences in different locations

### Expected Outcomes
- Understanding of where I spend most time on each platform
- Insights into location-based usage patterns
- Recommendations for better social media habits

## Privacy Note
- All raw data will be processed locally
- Only aggregated results will be shared
- No specific locations will be revealed
